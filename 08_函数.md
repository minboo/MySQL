## 函数


### 创建函数

学过的函数：LENGTH、SUBSTR、CONCAT等
语法：

	CREATE FUNCTION 函数名(参数名 参数类型,...) RETURNS 返回类型
	BEGIN
		函数体
	
	END

### 调用函数
	SELECT 函数名（实参列表）




### 函数和存储过程的区别

||关键字|调用语法|返回值|应用场景|
|:-:|:-:|:-:|:-:|:-:|
函数|FUNCTION|SELECT 函数()|只能是一个|一般用于查询结果为一个值并返回时，当有返回值而且仅仅一个
|存储过程|PROCEDURE|CALL 存储过程()|可以有0个或多个|一般用于更新
